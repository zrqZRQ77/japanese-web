### 1. 媒介契約（宅建業法）
不動産の売買や賃貸の仲介を不動産会社に依頼する契約です。3種類あり、ルールの違いがよく出題されます。

| 媒介契約の種類 | 複数の業者への依頼 | 自己発見取引（自分で買主を見つける） | レインズ（指定流通機構）への登録義務 | 契約の有効期間 |
| :--- | :--- | :--- | :--- | :--- |
| **専属専任媒介契約** | **不可** | **不可** | 契約締結日から**5日以内** | 最長3ヶ月 |
| **専任媒介契約** | **不可** | **可能** | 契約締結日から**7日以内** | 最長3ヶ月 |
| **一般媒介契約** | **可能** | **可能** | 登録義務なし（任意） | 法定の制限なし |

### 2. 手付金
売買契約の成立時に買主から売主に交付されるお金です。
- **解約手付（かいやくてつけ）**: 買主は「支払った手付金を**放棄**する」ことで、売主は「受け取った手付金の**倍額を返す（倍返し）**」ことで、理由を問わず契約を解除できます。
- **解除の期限**: 相手方が**「履行に着手するまで」**（例：売主が登記手続きを始めた、買主が中間金を支払ったなど）に限られます。

### 3. 宅地建物取引業法の重要ルール
- 宅建業者が売主となる場合、買主が宅建業者でないときは、手付金は売買代金の20％を超えることができません。
- 宅建業者は、契約前に重要事項説明書を交付し、宅地建物取引士が説明します。
- 売買契約が成立したら、契約内容を記載した書面を交付します。

### 4. 借地借家法
| 区分 | 主な内容 |
| :--- | :--- |
| 普通借地権 | 存続期間は原則30年以上。更新がある |
| 定期借地権 | 契約更新がなく、期間満了で終了する |
| 普通借家契約 | 正当事由がなければ貸主から更新拒絶しにくい |
| 定期借家契約 | 契約で定めた期間満了により終了する。書面等による契約が必要 |

> 不動産分野では、売買だけでなく賃貸借のルールも出題範囲です。
