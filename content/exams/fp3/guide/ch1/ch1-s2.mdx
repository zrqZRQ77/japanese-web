### 1. ライフプランニングの3大ツール
1. **ライフイベント表**: 家族のイベントと予想費用を時系列でまとめた表。
2. **キャッシュフロー表**: 現在の収支と将来の予測を一覧にした表。
3. **個人バランスシート**: ある時点における「資産」と「負債」の状況（※時価換算）。

### 2. キャッシュフロー表と個人バランスシート
- **年間収支**: 年間収入 － 年間支出。
- **金融資産残高**: 前年末金融資産残高 × (1 + 運用利率) + 当年年間収支。
- **可処分所得**: 年収 － 所得税・住民税 － 社会保険料。
- **純資産**: 資産合計 － 負債合計。

### 3. 資金計画のための6つの係数
- **終価係数**: 現在の元本から「将来の金額」を求める。
- **現価係数**: 将来の目標額から「現在必要な元本」を求める。
- **減債基金係数**: 将来の目標額から「毎年の積立額」を求める。
- **資本回収係数**: 現在の元本から「毎年の受取額・返済額」を求める。
- **年金終価係数**: 毎年一定額を積み立てた場合の将来の合計額を求める。
- **年金現価係数**: 毎年一定額を受け取るために現在必要な元本を求める。

| 目的 | 使う係数 |
| :--- | :--- |
| 今あるお金を将来いくらにするか | 終価係数 |
| 将来必要な金額から現在必要な元本を求める | 現価係数 |
| 目標額を作るための毎年の積立額を求める | 減債基金係数 |
| 元本を取り崩す場合の毎年の受取額を求める | 資本回収係数 |
| 毎年積み立てた結果の将来額を求める | 年金終価係数 |
| 毎年受け取るための現在必要額を求める | 年金現価係数 |
