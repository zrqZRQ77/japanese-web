# 法務

法務では、ITを利用する企業や個人が守るべき法律、契約、権利を学びます。知的財産権、個人情報保護、セキュリティ関連法規、労働契約、取引関連法規が中心です。

## 1. 知的財産権

| 権利 | 保護対象 | 注意点 |
|---|---|---|
| 特許権 | 発明 | 出願・登録が必要 |
| 実用新案権 | 物品の形状や構造の考案 | 出願・登録が必要 |
| 意匠権 | 物品などのデザイン | 出願・登録が必要 |
| 商標権 | 商品名、ロゴ、マーク | 出願・登録が必要 |
| 著作権 | 文章、音楽、画像、プログラムなど | 創作時に自動発生 |

プログラムのソースコードは著作権で保護されますが、アルゴリズムやプログラム言語そのものは著作権の保護対象ではありません。

## 2. 個人情報とセキュリティ関連法規

| 法律 | 内容 |
|---|---|
| 個人情報保護法 | 個人情報の取得、利用、保管、第三者提供などのルール |
| 不正アクセス禁止法 | なりすましや脆弱性を突いた不正侵入を禁止 |
| サイバーセキュリティ基本法 | 国や重要インフラなどのサイバーセキュリティ施策の基本 |
| 不正競争防止法 | 営業秘密の不正取得や不正利用などを禁止 |

## 3. 契約と労働関連法規

| 契約・法律 | 内容 |
|---|---|
| 請負契約 | 成果物の完成を約束する契約 |
| 準委任契約 | 業務の遂行を委任する契約 |
| 労働者派遣契約 | 派遣先が労働者に指揮命令する契約 |
| 労働基準法 | 労働時間、賃金、休暇などの基本ルール |
| 中小受託取引適正化法 | 受託事業者との取引適正化に関するルール |

## 4. その他の取引関連法規

| 法律 | 内容 |
|---|---|
| PL法 | 製造物の欠陥による損害について製造業者等の責任を定める |
| 特定商取引法 | 訪問販売や通信販売などの取引ルール |
| 電子契約関連 | 電子的な契約や署名の有効性に関係する |

## 試験での注意点

派遣契約では指揮命令権は派遣先にあります。請負契約では発注者が作業者へ直接指示すると偽装請負の問題になります。最新シラバスでは「下請法」ではなく「中小受託取引適正化法」として整理します。
