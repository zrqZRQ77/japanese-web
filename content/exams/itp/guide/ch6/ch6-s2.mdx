# コンプライアンスと標準化

コンプライアンスは法令遵守だけでなく、社会規範、社内規程、倫理を守ることまで含みます。標準化は、製品やサービス、手順を共通化し、品質や互換性を高める活動です。

## 1. コンプライアンス

| 用語 | 内容 |
|---|---|
| コンプライアンス | 法令、社内規程、社会規範を守ること |
| コーポレートガバナンス | 企業経営を監督し健全性を保つ仕組み |
| CSR | 企業の社会的責任 |
| 内部統制 | 業務を適正に行うための組織内の仕組み |
| 情報倫理 | 情報を扱う上で守るべき倫理 |

## 2. リスクと内部統制

企業は、不正、情報漏えい、会計ミス、法令違反を防ぐために内部統制を整備します。

| 対策 | 内容 |
|---|---|
| 職務分掌 | 1人に権限を集中させない |
| 承認手続 | 重要な処理に承認を必要とする |
| アクセス制御 | 必要な人だけが情報を扱えるようにする |
| ログ管理 | 操作記録を残し追跡できるようにする |
| 教育 | 社員へルールと倫理を周知する |

## 3. 標準化

| 種類 | 内容 |
|---|---|
| JIS | 日本産業規格 |
| ISO | 国際標準化機構による国際規格 |
| IEC | 電気・電子分野の国際規格 |
| IEEE | 電気・情報分野の標準化団体 |
| W3C | Web技術の標準化団体 |

## 4. 標準化のメリット

- 製品やサービスの互換性が高まる。
- 品質が安定する。
- 調達や取引がしやすくなる。
- 利用者が安心して選択できる。
- 国際的な連携がしやすくなる。

## 試験での注意点

コンプライアンスは「法律だけ守ればよい」という意味ではありません。社会的信用を守るため、社内規程、倫理、説明責任も含めて考えます。
