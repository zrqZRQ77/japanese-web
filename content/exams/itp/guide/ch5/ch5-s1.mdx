# 企業活動

企業活動では、組織がどのように価値を生み出し、業務を管理し、財務状況を把握するかを学びます。ITパスポートでは、業務改善、会計、経営指標、組織形態がよく問われます。

## 1. 企業と組織

| 用語 | 内容 |
|---|---|
| 株式会社 | 株式を発行して資金を集める会社 |
| 組織図 | 指揮命令系統や部署の関係を示す図 |
| 職能別組織 | 営業、製造、経理など機能別に分ける組織 |
| 事業部制組織 | 製品や地域ごとに事業部を分ける組織 |
| マトリックス組織 | 機能別とプロジェクト別などを組み合わせる組織 |

## 2. 業務改善

| 手法 | 内容 |
|---|---|
| PDCA | 計画、実行、評価、改善を繰り返す |
| BPR | 業務プロセスを抜本的に見直す |
| BPM | 業務プロセスを継続的に管理・改善する |
| KGI | 最終目標を示す指標 |
| KPI | 目標達成までの進捗を測る指標 |

## 3. 会計と財務

| 書類・指標 | 内容 |
|---|---|
| 貸借対照表 | 一時点の資産、負債、純資産を示す |
| 損益計算書 | 一定期間の収益、費用、利益を示す |
| キャッシュフロー計算書 | 現金の増減を示す |
| 損益分岐点 | 利益が0になる売上高 |
| ROI | 投資に対する利益の割合 |
| ROE | 自己資本に対する利益の割合 |

## 4. 業務で使う分析

- ABC分析: 重要度や売上構成により重点管理する。
- パレート図: 重要な原因を見つける。
- 散布図: 2つの値の関係を確認する。
- ヒストグラム: データの分布を確認する。

## 試験での注意点

財務諸表は、貸借対照表が「時点」、損益計算書が「期間」です。KGIは最終目標、KPIは途中の管理指標として区別します。
