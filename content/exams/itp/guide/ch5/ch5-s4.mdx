# ビジネスインダストリ

ビジネスインダストリでは、業界や社会で使われる代表的な情報システム、電子商取引、AI、IoT、DXなどを学びます。

## 1. 電子商取引

| 用語 | 内容 |
|---|---|
| BtoC | 企業から個人への取引 |
| BtoB | 企業間取引 |
| CtoC | 個人間取引 |
| EC | インターネット上で商品やサービスを売買する |
| EDI | 企業間の取引データを電子的に交換する |

## 2. 業務システム

| システム | 内容 |
|---|---|
| POS | 販売時点の情報を収集する |
| CAD | 設計を支援する |
| CAM | 製造を支援する |
| FA | 工場の自動化 |
| GPS連携システム | 位置情報を活用する |

## 3. デジタル社会の技術

| 技術 | 内容 |
|---|---|
| AI | 学習や推論により判断・予測を支援する |
| 生成AI | 文章、画像、音声、コードなどを生成する |
| IoT | モノをネットワークにつなぎ情報を活用する |
| ビッグデータ | 大量・多様・高速に発生するデータ |
| RPA | 定型的な事務作業を自動化する |
| ブロックチェーン | 改ざん耐性の高い分散台帳技術 |

## 4. DX

DXは、単に紙の業務を電子化することではありません。デジタル技術とデータを活用して、業務、組織、ビジネスモデル、顧客体験を変革することです。

## 試験での注意点

デジタイゼーションはアナログ情報のデジタル化、デジタライゼーションは業務プロセスのデジタル化、DXはビジネス全体の変革として区別します。
