# 技術戦略マネジメント

技術戦略マネジメントでは、新しい技術をどのように事業価値へつなげるかを学びます。研究開発、イノベーション、知的財産、技術ポートフォリオが中心です。

## 1. 技術開発戦略

| 用語 | 内容 |
|---|---|
| 研究開発 | 新しい知識や技術を生み出す活動 |
| 技術ロードマップ | 技術の発展や導入計画を時系列で示す |
| コアコンピタンス | 他社がまねしにくい中核的な強み |
| オープンイノベーション | 外部の技術や知識も活用して革新する |

## 2. イノベーション

| 種類 | 内容 |
|---|---|
| プロダクトイノベーション | 新しい製品やサービスを生み出す |
| プロセスイノベーション | 生産や業務の方法を革新する |
| 破壊的イノベーション | 既存市場の構造を変える革新 |
| 持続的イノベーション | 既存製品を継続的に改善する |

## 3. 技術と知的財産

新技術は、特許、ノウハウ、営業秘密などとして保護し、事業戦略と結び付けて活用します。技術情報を外部と共有する場合は、契約や秘密保持の管理が必要です。

## 4. デジタル技術の活用

- AIによる予測や自動化。
- IoTによる設備や商品の状態把握。
- ビッグデータ分析による需要予測。
- クラウドによる迅速なサービス展開。
- 生成AIによる文章作成、要約、アイデア出しの支援。

## 試験での注意点

技術戦略は「新しい技術を知っているか」だけではなく、「技術を事業価値へどうつなげるか」が問われます。研究開発、知財、経営戦略をセットで考えます。
