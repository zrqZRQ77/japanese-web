# 経営戦略マネジメント

経営戦略マネジメントでは、企業が競争優位を作るための分析手法やマーケティング、経営管理システムを学びます。

## 1. 経営環境分析

| 手法 | 内容 |
|---|---|
| SWOT分析 | 強み、弱み、機会、脅威で分析する |
| 3C分析 | 顧客、自社、競合を分析する |
| PEST分析 | 政治、経済、社会、技術の外部環境を分析する |
| ファイブフォース分析 | 業界の競争要因を分析する |

## 2. 事業戦略

| 用語 | 内容 |
|---|---|
| PPM | 市場成長率と市場占有率で事業を分類する |
| 差別化戦略 | 他社と異なる価値で競争する |
| コストリーダーシップ | 低コストで競争優位を作る |
| 集中戦略 | 特定市場に経営資源を集中する |

## 3. マーケティング

| 用語 | 内容 |
|---|---|
| STP | セグメンテーション、ターゲティング、ポジショニング |
| 4P | 製品、価格、流通、販売促進 |
| CRM | 顧客関係を管理し長期的な関係を築く |
| CS | 顧客満足 |
| ブランド戦略 | 製品や企業の価値を高める戦略 |

## 4. 経営管理システム

| システム | 内容 |
|---|---|
| ERP | 企業資源を統合的に管理する |
| SCM | 調達から販売までの供給連鎖を管理する |
| CRM | 顧客情報や営業活動を管理する |
| SFA | 営業活動を支援する |
| BI | データを分析し意思決定を支援する |

## 試験での注意点

SWOTは内部環境と外部環境を分けて考えます。強み・弱みは自社内部、機会・脅威は外部環境です。PPMは「市場成長率」と「市場占有率」の2軸です。
