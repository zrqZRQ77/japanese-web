# 情報メディア

情報メディアは、文字、音声、画像、動画などをデジタルデータとして扱う分野です。ITパスポートでは、ファイル形式、圧縮、解像度、色の表現、マルチメディアの基本が問われます。

## 1. 画像・音声・動画の基本

| メディア | 主な要素 | 例 |
|---|---|---|
| 画像 | 画素、解像度、色数 | 写真、図表 |
| 音声 | サンプリング周波数、量子化ビット数 | 音楽、通話 |
| 動画 | フレームレート、圧縮方式 | 配信動画、会議録画 |

## 2. 代表的なファイル形式

| 形式 | 種類 | 特徴 |
|---|---|---|
| JPEG | 画像 | 写真向き。非可逆圧縮 |
| PNG | 画像 | 透過や劣化なし保存に向く |
| GIF | 画像 | 色数が少なく簡単なアニメーションに対応 |
| SVG | 画像 | 拡大しても劣化しにくいベクタ形式 |
| MP3 | 音声 | 音楽配信で広く使われる |
| MP4 | 動画 | 動画配信で広く使われる |

## 3. 圧縮方式

| 種類 | 内容 | 例 |
|---|---|---|
| 可逆圧縮 | 元のデータに完全に戻せる | ZIP、PNG |
| 非可逆圧縮 | 一部情報を捨てて容量を小さくする | JPEG、MP3 |

## 4. 色と表示

- **RGB**: 赤、緑、青の光で色を表す方式。ディスプレイで使う。
- **CMYK**: シアン、マゼンタ、イエロー、黒で色を表す方式。印刷で使う。
- **解像度**: 画像の細かさ。一般に高いほど精細だが容量も増える。
- **ピクセル**: 画像を構成する最小単位。

## 試験での注意点

写真にはJPEG、透過画像にはPNG、図形やロゴにはSVGが向く、という使い分けはよく問われます。圧縮では、可逆と非可逆の違いを必ず押さえましょう。
