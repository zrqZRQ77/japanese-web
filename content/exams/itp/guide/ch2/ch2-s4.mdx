# ネットワーク

ネットワークは、コンピュータ同士がデータをやり取りする仕組みです。ITパスポートでは、LANとWAN、TCP/IP、IPアドレス、DNS、代表的な通信プロトコル、無線通信がよく出ます。

## 1. ネットワークの種類

| 種類 | 内容 |
|---|---|
| LAN | 社内や家庭など限られた範囲のネットワーク |
| WAN | 遠隔地のLAN同士を結ぶ広域ネットワーク |
| インターネット | 世界中のネットワークを相互接続したもの |
| VPN | 公衆回線上に仮想的な専用線を作る技術 |

## 2. TCP/IPとIPアドレス

| 用語 | 内容 |
|---|---|
| TCP/IP | インターネットで標準的に使われる通信規約 |
| IPアドレス | ネットワーク上の機器を識別する番号 |
| グローバルIPアドレス | インターネット上で一意のアドレス |
| プライベートIPアドレス | 組織内や家庭内で使うアドレス |
| NAT | プライベートIPとグローバルIPを変換する技術 |

## 3. 代表的なプロトコル

| プロトコル | 用途 |
|---|---|
| HTTP | Webページの通信 |
| HTTPS | 暗号化されたWeb通信 |
| SMTP | メール送信 |
| POP / IMAP | メール受信 |
| DNS | ドメイン名とIPアドレスの対応付け |
| DHCP | IPアドレスなどを自動設定 |
| FTP | ファイル転送 |

## 4. 無線通信

| 種類 | 用途 |
|---|---|
| Wi-Fi | 無線LAN |
| Bluetooth | 近距離の機器接続 |
| NFC | 非接触ICカードなど |
| LTE / 5G | 携帯電話網を使う通信 |

## 試験での注意点

DNSは名前解決、DHCPは自動設定、HTTPSは暗号化されたWeb通信です。似た略語が多いので、何を実現する技術かで整理しましょう。
